#  Titanic Survival Prediction (Data Science Project)

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning. It demonstrates a complete data science workflow including preprocessing, visualization, feature engineering, model building, and evaluation.

##  Objective
To analyze the Titanic dataset and build a predictive model based on passenger features.

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn

## Dataset
- Titanic dataset (CSV)
- Features:
  - Age
  - Sex
  - Pclass
  - Fare
  - FamilySize

## Workflow
1. Data Cleaning
2. Data Visualization
3. Feature Engineering
4. Model Building (Logistic Regression)
5. Evaluation (Accuracy & Confusion Matrix)

## Results
- Accuracy: ~75% – 85%

##  How to Run
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
python titanic_project.py
```
